Project Overview

This is a rule-based algorithmic trading system that trades XAUUSD and EURUSD pairs using Python and Pine Script. It executes trades based on Smart Money Concepts (SMC), Order Blocks, and Time-based Killzones to identify high-probability setups while eliminating emotional decision-making. The strategy focuses on institutional-level analysis to capture market inefficiencies across multiple timeframes.
